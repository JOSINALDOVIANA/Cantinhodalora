PORT=>porta onde roda o backend
SERVER_URL=>todas as imagens utilizam este dominio para o frontend acessar suas imagens exemplo 'https://www.cantinhodalora.info/'
DB_HOST=>ip do banco de dados
DB_PORT=>porta do banco de dados
DB_USER=>usuario do banco de dados
DB_PASS=>senha do banco de dados
DB_NAME=>nome do banco de dados
DB_DIALECT=>tipo do banco de dados
DB_POOL_MAX=>maximo de conexoes com o banco de dados
DB_POOL_MIN=>minimo de conexoes com o banco de dados
DB_POOL_ACQUIRE=>tempo maximo de espera para adquirir uma conexao com o banco de dados
DB_POOL_IDLE=>tempo maximo de inatividade de uma conexao com o banco de dados
NODE_ENV=>ambiente de desenvolvimento ou produção (development, production)

ACCESS_SECRET=>segredo jwt para tokens de acesso
REFRESH_SECRET=>segredo jwt para tokens de renovacao
