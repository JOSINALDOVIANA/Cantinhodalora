# Cantinhodalora

Projeto particular de um e-commerce para Vendas de produtos .

## 🚀 Funcionalidades

- Cadastro e gerenciamento de produtos
- Cadastro e gerenciamento de categorias
- Cadastro e gerenciamento de usuários
- Cadastro e gerenciamento de clientes
- Upload e gerenciamento de imagens
- Login e logout
- Renovação de tokens

## 🛠️ Tecnologias

### Backend

- Node.js
- Express
- JWT
- Cookie-parser
- Request-ip
- Cors

### Frontend

- React
- TypeScript
- Tailwind CSS
- Axios
- React Router
- Cookie-parser

## 🔌 Instalação

### Backend

```bash
cd backend
npm install
yarn dev
```

### Frontend

```bash
cd front
npm install
yarn start 
```
